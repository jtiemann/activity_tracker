Chart Debug Helper

This page provides tools to debug weekly distribution chart issues.

Instructions:

  1. Open the main Activity Tracker app in another tab
  2. Login and select the "Pull-ups" activity
  3. Open your browser's developer console (F12)
  4. Run the diagnostic commands below
  5. Check the console output for errors or issues

Diagnostic Commands

Debug Steps

1. Select "Pull-ups" activity in the app
2. Check if any logs appear in the history panel
3. Open console (F12) and paste: window.showWeeklyChart()
4. Check for any errors or warning messages
5. Paste: console.log(window.activityCharts.data)
6. Check if logs array has any data
7. Check if the logs have valid logged_at dates

Common Issues

  • No log data available for "Pull-ups" activity
  • Invalid date formats in log entries
  • Chart canvas element not properly initialized
  • Chart library not loaded correctly
